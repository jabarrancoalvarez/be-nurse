Vas a realizar un refinamiento visual completo del proyecto be-nurse que ya existe en este directorio. NO toques el backend ni la logica de negocio. Solo el frontend Angular. El objetivo es transformar el frontend en una experiencia visual espectacular, de nivel award-winning, con animaciones fluidas, micro-interacciones en todo y un diseño que impacte desde el primer segundo.

PASO 1 — INSTALAR DEPENDENCIAS

Dentro de la carpeta frontend, ejecuta:

npm install gsap @types/gsap
npm install lenis
npm install swiper
npm install countup.js
npm install splitting
npm install vanta
npm install three @types/three
npm install @angular/animations

PASO 2 — CONFIGURACION GLOBAL DE ANIMACIONES

Crea el archivo frontend/src/app/core/animations/gsap.config.ts con el siguiente contenido: importa gsap y ScrollTrigger de gsap/ScrollTrigger, registra el plugin con gsap.registerPlugin(ScrollTrigger), configura ScrollTrigger.defaults con markers false y toggleActions "play none none reverse", exporta gsap y ScrollTrigger como named exports para usarlos en toda la app.

Crea frontend/src/app/core/animations/lenis.config.ts: inicializa Lenis con duration 1.4, easing funcion t => Math.min(1, 1.001 - Math.pow(2, -10*t)), direction vertical, gestureDirection vertical, smooth true, smoothTouch false, touchMultiplier 2. Conecta Lenis con GSAP ticker: gsap.ticker.add(time => lenis.raf(time * 1000)). Exporta la instancia de lenis.

Crea frontend/src/app/core/animations/animations.ts con estas animaciones reutilizables exportadas como funciones:

- fadeInUp(element, delay=0): gsap.fromTo element desde opacity 0 y y 60 hasta opacity 1 y y 0, duration 0.9, ease "power3.out", delay
- fadeInLeft(element, delay=0): igual pero desde x -60
- fadeInRight(element, delay=0): igual pero desde x 60
- staggerFadeIn(elements, stagger=0.15): gsap.fromTo con stagger, cada elemento desde opacity 0 y y 40
- scaleIn(element, delay=0): desde scale 0.85 y opacity 0 hasta scale 1 y opacity 1
- textReveal(element): split el texto en chars y anima cada char con stagger 0.03 desde y 100 y opacity 0, clip con overflow hidden en el parent
- counterAnimation(element, endValue): usa CountUp con duration 2.5 y separator punto
- parallaxElement(element, speed=0.5): ScrollTrigger con scrub true y movimiento vertical segun speed
